# Graph Theory Teaching Aid

An interactive desktop application for learning and visualising 
core graph theory algorithms through real-time feedback and 
psychologically-safe UX design.

## Algorithms Implemented
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- TSP Brute Force
- Nearest Neighbour Algorithm

## Tech Stack
- Visual Basic (.NET)
- Graph visualisation engine
- Cross-platform desktop support

## Features
- Real-time node/edge visualisation
- Step-by-step algorithm walkthrough
- Non-judgmental guided correction flows
- Adaptive interaction design for learning retention
